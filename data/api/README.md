backend.

- raw resale data for API queries
- saved model metrics and artifacts
- school-specific RDD outputs
- town premium outputs
- diagnostics outputs
- benchmark outputs

The API also keeps a copy of the engineered feature table here for prediction
defaults and feature engineering:

- `resale_flats_with_school_buffer_counts_with_walkability.csv`
