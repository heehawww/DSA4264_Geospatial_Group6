# DSA4264 Geospatial Group 6
This project aims to estimate the effect of being located near a “good” primary school on HDB resale prices.

## Hedonic model

A baseline hedonic regression pipeline is available in [hedonic_model/README.md](/Users/yongjun/Documents/New%20project/DSA4264_Geospatial_Group6/hedonic_model/README.md).

Install the Python dependencies first:

```bash
pip install -r requirements.txt
```

Run from the repository root:

```bash
python3 hedonic_model/train_hedonic_model.py
```
